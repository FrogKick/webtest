# Word Trainer

Простое приложение на Python для изучения новых слов.

## Что умеет

- Главное меню с быстрым обзором прогресса.
- Библиотека списков слов.
- Добавление слов: открытая часть `слово` и закрытая часть `значение`.
- Тренировка карточками: сначала видно слово, затем открывается значение.
- Статистика по словам, спискам, повторениям и точности.
- Профиль ученика: имя, цель, языковая пара и дневная цель.
- Экспорт данных в JSON.
- Хранение всей информации в SQLite-файле `data.db`.
- Адаптивный интерфейс: работает на ПК, планшете и телефоне через браузер.

## Быстрый запуск на ПК

### Windows

1. Установите Python 3.10 или новее.
2. Распакуйте архив.
3. Дважды нажмите `start_windows.bat`.
4. Откройте в браузере адрес:

```text
http://127.0.0.1:5000
```

### Linux или macOS

```bash
chmod +x start_unix.sh
./start_unix.sh
```

Затем откройте:

```text
http://127.0.0.1:5000
```

## Запуск вручную

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python app.py
```

Для Windows активация окружения:

```bat
.venv\Scripts\activate
```

## Как открыть на телефоне

1. Запустите приложение на ПК.
2. Убедитесь, что телефон и ПК подключены к одной Wi-Fi сети.
3. Узнайте IP-адрес ПК в локальной сети.
4. На телефоне откройте:

```text
http://IP_АДРЕС_ПК:5000
```

Пример:

```text
http://192.168.1.25:5000
```

## Где хранятся данные

Все данные лежат в файле:

```text
data.db
```

Если нужно перенести словарь на другой компьютер, скопируйте папку проекта вместе с этим файлом.

## Структура

```text
word_trainer/

  data.db
  requirements.txt
  start_windows.bat
  start_unix.sh

  static/
```

## Примечание

При первом запуске приложение автоматически создаёт базу данных и добавляет стартовый список слов для проверки.
